## Описание

Training Planner - это GUI-приложение для планирования и учёта тренировок.

**Возможности:**
- Добавление тренировок (дата, тип, длительность)
- Просмотр списка тренировок в таблице
- Фильтрация по типу и дате
- Редактирование и удаление записей
- Автоматическое сохранение в JSON
- Статистика тренировок
