# Weather Diary - Дневник погоды

**Автор:** Uraltsev Aleksej

---

## Описание

Weather Diary - это GUI-приложение для ведения дневника наблюдений за погодой.

**Возможности:**
- Добавление записей (дата, температура, описание, осадки)
- Просмотр списка записей в таблице
- Фильтрация по дате (конкретный день)
- Фильтрация по температуре (выше указанного значения)
- Редактирование и удаление записей
- Автоматическое сохранение в JSON
- Статистика (средняя температура)

---

## Запуск

```bash
python weather_diary.py
